## Version Policy

Kida follows [Semantic Versioning](https://semver.org/):

- **MAJOR**: Breaking API changes
- **MINOR**: New features (backward compatible)
- **PATCH**: Bug fixes (backward compatible)

## Installation

```bash
# Latest stable
pip install kida

# Specific version
pip install kida==0.1.0
```
